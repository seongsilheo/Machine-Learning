## 1.1.2 머신러닝 문제의 분류

1) Supervised learning(지도 학습) : 입력에 대한 적절한 출력을 찾는 문제

2) Unsupervised learning(비지도 학습) : 입력 정보의 특징을 찾는 문제

3) Reinforcement learning(강화 학습) : 전체적인 결과가 가장 좋은 행동을 찾는 문제(장기, 체스 등)
